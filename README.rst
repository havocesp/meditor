=========
RSTEditor
=========
RSTEditor is a editor for ReStructedText. It support preview with html.

Requirement
===========
Install them on all platform:

+ docutils
+ Pygments
+ pyqt4

Py2exe
=======
To make a Py2exe package, you need copy docutils and pygments files into ``dist`` directory manually.

Template
========
template::

    skeleton.rst
